# Breast-Cancer-ML-Prediction-Model
Machine learning model to predict breast cancer diagnosis using clinical features (Python, Scikit-learn)
## Project Overview
This project applies machine learning techniques to predict whether a tumor is malignant or benign using clinical features.
## Dataset
- Breast Cancer Wisconsin Dataset
- Features include radius, texture, perimeter, area, and smoothness
## Methods
- Data preprocessing using Pandas
- Model training using Scikit-learn
- Algorithms: Logistic Regression, Support Vector Machine
## Results
- High classification accuracy
- Evaluation using confusion matrix and ROC curve
## How to Run
1. Open the notebook
2. Add your Kaggle API key (not included for security)
3. Run all cells
