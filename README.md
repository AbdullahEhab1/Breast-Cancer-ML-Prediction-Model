# Breast Cancer Prediction Using Machine Learning
This project applies machine learning to classify breast tumors as malignant or benign using clinical diagnostic features.
## Biological Relevance
Breast cancer is one of the most common cancers worldwide. Early detection using computational models can support clinical decision-making and improve patient outcomes.
## Methods
- Data preprocessing and cleaning
- Feature scaling (StandardScaler)
- Logistic Regression model
- Train-test split (75/25)
## Results
High classification accuracy
Performance evaluated using confusion matrix and accuracy score

Python
Pandas
Scikit-learn
